# MongoDB Connection - Spring Data MongoDB Basics

## What this demonstrates
- `spring-boot-starter-data-mongodb` dependency
- `@Document` annotation to map a class to a MongoDB collection
- `MongoRepository` interface for built-in CRUD operations
- Custom query methods derived from method names (`findByDepartment`, `findByNameContainingIgnoreCase`)
- MongoDB configuration in `application.properties`

## Prerequisites
MongoDB must be running on localhost:27017. Start it with:
```bash
mongod
# or if using Homebrew on Mac:
brew services start mongodb-community
```

## How to run
```bash
cd mongo-connection
mvn spring-boot:run
```
The application starts on port 8080. The database `demo_db` and collection `students` are created automatically.

## Endpoints to test

```bash
# Create a student
curl -X POST http://localhost:8080/api/students \
  -H "Content-Type: application/json" \
  -d '{"name":"Krushi","rollNumber":"21CS001","department":"CSE","email":"krushi@example.com"}'
# Expected: 201 Created with student JSON (id auto-generated by MongoDB)

# Create another student
curl -X POST http://localhost:8080/api/students \
  -H "Content-Type: application/json" \
  -d '{"name":"Ravi","rollNumber":"21EC002","department":"ECE","email":"ravi@example.com"}'

# Get all students
curl http://localhost:8080/api/students
# Expected: JSON array of all students

# Get by id (use the id from the POST response)
curl http://localhost:8080/api/students/{id}

# Find by department
curl http://localhost:8080/api/students/department/CSE
# Expected: All CSE students

# Search by name
curl http://localhost:8080/api/students/search?name=krushi
# Expected: Students whose name contains "krushi" (case-insensitive)

# Update a student
curl -X PUT http://localhost:8080/api/students/{id} \
  -H "Content-Type: application/json" \
  -d '{"name":"Krushi Raj","rollNumber":"21CS001","department":"CSE","email":"krushi.raj@example.com"}'

# Delete a student
curl -X DELETE http://localhost:8080/api/students/{id}
# Expected: 204 No Content
```
